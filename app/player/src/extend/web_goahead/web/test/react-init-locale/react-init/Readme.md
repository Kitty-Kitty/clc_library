react-starter-kit
====

### 组件

1. React
2. React-router
3. Redux
4. webpack

### 运行

1. git clone
2. npm install
3. node server.js
4. open browser navigate to http://localhost:3000


### Tips

* 可以使用 webpack-bundle-analyzer 分支最终webpack打包的内容
* proxy 转发参看 https://medium.com/@jamsesso/webpack-proxy-with-http-proxy-middleware-14331e7562d2#.xzzsivf2r
